# 手動テストチェックリスト v0.2

v0.1 のチェックリストをベースに、v0.2 機能（応援トーン選択・表示頻度選択・設定保存・称号）の確認項目を追加したもの。
公開URL: https://2727-art.github.io/anju-training/

## アクセス

- [ ] iPhone Safari で Pages URL を開ける
- [ ] Android Chrome で Pages URL を開ける
- [ ] PC Chrome で Pages URL を開ける

## 動画選択

- [ ] 縦長動画を選択できる
- [ ] 横長動画が弾かれる（縦長のみ対応のエラー表示）
- [ ] 正方形に近い動画が弾かれる
- [ ] 15秒未満動画で警告（簡易セッションの注意）が出る

## セッション進行

- [ ] 30〜60秒動画でセッション完走できる
- [ ] 120秒以上動画でHUDが破綻しない
- [ ] 動画再生中にHUDが中央を邪魔しすぎない
- [ ] 応援メッセージが連続で同じにならない
- [ ] CLEAR / FEVER / LAST PUSH の表示タイミングが不自然でない

## v0.2: 応援トーン選択

- [ ] 応援トーン「おまかせ」が選択できる
- [ ] 応援トーン「やさしめ」が選択できる
- [ ] 応援トーン「元気」が選択できる
- [ ] 応援トーン「集中」が選択できる
- [ ] 応援トーン「しずか」が選択できる
- [ ] 選択したトーンに合う雰囲気のメッセージが中心に出る

## v0.2: 表示頻度選択

- [ ] 表示頻度「少なめ」が選択できる
- [ ] 表示頻度「ふつう」が選択できる
- [ ] 表示頻度「多め」が選択できる
- [ ] 「少なめ」と「多め」でメッセージ量の違いが体感できる

## v0.2: 設定保存

- [ ] 選択したトーンがリロード後も復元される
- [ ] 選択した頻度がリロード後も復元される
- [ ] 「もう一度プレイ」で同じ設定が維持される
- [ ] 動画ファイル名が localStorage に保存されない（DevTools → Application → Local Storage で確認）
- [ ] Blob URL が localStorage に保存されない
- [ ] 動画メタデータ（幅・高さ・尺）が localStorage に保存されない

## 画面遷移・結果

- [ ] 動画終了後に結果画面へ遷移する
- [ ] 結果画面に称号が表示される
- [ ] 「もう一度プレイ」で同じ動画を再開できる
- [ ] 「別の動画を選ぶ」で動画選択画面に戻る
- [ ] サバイバルモードが選択不可（Coming soon）のまま

## レイアウト・アクセシビリティ

- [ ] safe-area がある端末で上下のHUD・ボタンが隠れない
- [ ] prefers-reduced-motion 環境で過剰演出にならない
- [ ] 端末を縦向きで使った時にレイアウトが崩れない

## 技術・プライバシー

- [ ] コンソールにエラーが出ない
- [ ] 外部通信が発生しない（初回ロードの静的アセット以外のリクエストがない）
- [ ] 動画はアップロードされない
