# Release Notes v0.2

## 概要

v0.2では、応援モードのメッセージ体験を強化しました。

## 追加

- 応援メッセージ231件（v0.1: 120件 → 約2倍）
- 応援トーン選択（おまかせ / やさしめ / 元気 / 集中 / しずか）
- メッセージ表示頻度選択（少なめ / ふつう / 多め）
- 設定保存（localStorage。トーンと頻度のみ。動画情報は保存しない）
- 称号36件（トーン・頻度・尺・完走のしかたに応じて選定）
- メッセージ品質テスト強化（件数・重複・NGワード・フェーズ/トーン分布）
- 設定保存仕様・メッセージレビュー・リリース手順のドキュメント

## 変更

- メッセージ選択ロジックの改善（トーン指定の優先・フェーズ適合・重複回避）
- 結果画面の称号バリエーション拡充（10件 → 36件、コンテキスト連動）
- READMEとドキュメント更新

## 変えていないこと

- 動画アップロードなし
- サーバーなし
- 外部APIなし
- 動作解析なし
- 実測成績表示なし
- サバイバルモード無効（Coming soon のまま）

## 既知の制限

- 実機確認は端末ごとに手動で必要（実機確認は一部未完了）
- 動画再生可否はブラウザの対応形式に依存
- 横長・正方形動画は非対応
- サバイバルモードはComing soon

## 確認コマンド

- npm run lint
- npm run test -- --run
- npm run build
- npm run ci
