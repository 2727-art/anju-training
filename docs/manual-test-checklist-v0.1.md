# 手動テストチェックリスト v0.1

自動テストでは確認しづらい、実動画・実ブラウザでの確認項目。
v0.1 Release Candidate を公開する前に一通り確認する。

## 動画選択

- [ ] iPhone Safari で縦長動画を選択できる
- [ ] Android Chrome で縦長動画を選択できる
- [ ] PC Chrome で縦長動画を選択できる
- [ ] 15秒未満の動画で警告（簡易セッションの注意）が出る
- [ ] 横長動画を選ぶと弾かれる（縦長のみ対応のエラー表示）
- [ ] 正方形に近い動画（例: 1080x1080）を選ぶと弾かれる

## セッション進行

- [ ] 15〜30秒の動画で短尺セッションが進む（ミッション2〜4個）
- [ ] 30〜60秒の動画で通常セッションが進む（ミッション4〜6個）
- [ ] 120秒以上の動画でHUDが破綻しない（残り時間・ゲージ・ミッションが正常）
- [ ] 動画再生中にHUDが中央を邪魔しすぎない
- [ ] 応援メッセージが連続で同じにならない
- [ ] CLEAR / FEVER / LAST PUSH / FINISH の表示タイミングが不自然でない
- [ ] 再生・一時停止・リセットが動作する

## 画面遷移

- [ ] 動画終了後に結果画面へ遷移する
- [ ] 結果画面に完走時間・ミッション数・応援メッセージ数・最大コンボ・称号が出る
- [ ] 「もう一度プレイ」で同じ動画を再開できる
- [ ] 「別の動画を選ぶ」で動画選択に戻れる
- [ ] サバイバルモードが Coming soon（選択不可）のままである

## レイアウト・アクセシビリティ

- [ ] 端末を縦向きで使った時にレイアウトが崩れない
- [ ] safe-area がある端末（ノッチ付きiPhone等）で上下のHUD・ボタンが隠れない
- [ ] prefers-reduced-motion 環境で過剰な演出にならない
- [ ] ボタンのタップ領域が十分に大きい

## 技術・プライバシー

- [ ] ブラウザのコンソールにエラーが出ない
- [ ] 動画はアップロードされない（DevTools の Network で動画送信がないこと）
- [ ] 外部通信が発生しない（初回ロードの静的アセット以外のリクエストがないこと）
- [ ] GitHub Pages 公開URL（https://2727-art.github.io/anju-training/）で直接開いて動作する
